Marze solver

The Maze Solver Project aims to develop a program capable of efficiently navigating through mazes from a starting point to a destination. The project employs various algorithms, including Depth-First Search (DFS) and Breadth-First Search (BFS), to explore and solve mazes. These algorithms differ in their exploration strategies, with DFS prioritizing depth-first exploration and BFS focusing on breadth-first traversal.

Mini Viting System

A mini voting system project in C involves creating a program where users can vote for options, tallying the votes, and displaying the results. It includes features like a user interface for voting, counting the votes, and error handling. Optionally, data persistence can be implemented for storing vote counts.
